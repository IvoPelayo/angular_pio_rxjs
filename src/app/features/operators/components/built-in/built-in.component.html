Operators

Ahora que ya tenemos una idea básica de como trabajar con Observables, podemos dar paso a una de las principales ventajas: los operadores. Estas funciones permiten expandir comportamiento, combinar, transformar observables dando la posiblidad de realizar operaciones complejas, capturar errores durante el proceso, reutilizar implementaciones y mucho más.

Dentro de la documentación de la librería encontramos una herramienta muy útil para saber cuál operador es el que más se adecua a nuestro requerimiento: Arbol de decisión de operadores
