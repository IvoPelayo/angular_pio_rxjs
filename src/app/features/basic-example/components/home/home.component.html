Introducción

Es una librería de javascript que permite gestionar eventos de todo tipo: Llamadas a la API, cambios de elementos de DOM, inputs, Angular forms, entre otros. Tambíen facilita la manipulación de estos eventos, como puede ser filtrado, mapeos, delays, y más.

Elementos

  • Observable: Una colección de eventos que se pueden emitir en algún momento.
  • Observer: Un objeto que escucha el flujo de datos y puede actuar sobre los valores que éste emite.
  • Subscription: Representa la ejecución de un observable y permite cancelarla.
  • Operador: Función para manipular los eventos siguiendo los principios de la programación funcional.
